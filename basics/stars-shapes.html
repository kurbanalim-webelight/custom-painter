How a star is different from a polygon

A star has pointed tips (like the tips of a star) and inward dips between them. So we need two kinds of points: outer points (the tips, far from the centre) and inner points (the dips, closer to the centre). We place them one after the other in an alternating way: outer, inner, outer, inner, ‚Ä¶ That‚Äôs what gives the star shape.

Outer and inner radius

We use two circles with the same centre but different sizes:

  • Outer radius ‚Äî distance from centre to the tips of the star. All tip points lie on this circle.
  • Inner radius ‚Äî distance from centre to the dips (the inward corners). All dip points lie on this smaller circle.

So we have two radii: outerRadius and innerRadius. The more the inner radius is smaller than the outer, the ‚Äúsharper‚Äù and more star-like the shape. If inner and outer were equal, we would get a normal polygon. So the difference between these two radii is what creates the star look.

Alternating points

We have a fixed number of points (e.g. 5 for a 5-pointed star). For each point we need an angle, same as in a polygon: angle step = 360¬∞ √∑ number of points (in radians: 2œÄ / points). So point 0 at angle 0, point 1 at angleStep, point 2 at 2*angleStep, etc.

The only twist: we alternate the radius. For point 0 we use outer radius, for point 1 we use inner radius, for point 2 we use outer again, then inner, and so on. So:

  • Even index (0, 2, 4, ‚Ä¶) ‚Üí use outerRadius ‚Üí tip of the star
  • Odd index (1, 3, 5, ‚Ä¶) ‚Üí use innerRadius ‚Üí dip between tips

Each point is still: Offset(centerX + radius * cos(angle), centerY + radius * sin(angle)), but radius is either outerRadius or innerRadius depending on whether the index is even or odd. We add these points to the path in order and close it ‚Äî that‚Äôs our star.

üí° In short: Same angle step as a polygon. At each angle, use outer radius for even-index points (tips) and inner radius for odd-index points (dips). Join the points in order and close the path.

Example code

Below, starPath is defined inside paint() and used to draw a 5-pointed star at the centre of the canvas. The GIF shows the result.

@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()
    ..style = PaintingStyle.stroke
    ..strokeWidth = 3
    ..color = Colors.blue;

  Path starPath({
    required Offset center,
    required int points,
    required double outerRadius,
    required double innerRadius,
    double startAngle = -math.pi / 2, // top tip first
  }) {
    final path = Path();
    final angleStep = math.pi / points; // half turn / points
    for (var i = 0; i < 2 * points; i++) {
      final angle = startAngle + i * angleStep;
      final radius = i.isEven ? outerRadius : innerRadius;
      final x = center.dx + radius * math.cos(angle);
      final y = center.dy + radius * math.sin(angle);
      if (i == 0) {
        path.moveTo(x, y);
      } else {
        path.lineTo(x, y);
      }
    }
    path.close();
    return path;
  }

  // Usage: 5-pointed star
  canvas.drawPath(
    starPath(
      center: Offset(size.width / 2, size.height / 2),
      points: 5,
      outerRadius: 60,
      innerRadius: 25,
    ),
    paint,
  );
}

5-pointed star drawn with starPath (outer radius 60, inner radius 25).

We go through 2 * points positions (each tip and each dip) and use i.isEven ? outerRadius : innerRadius so tips and dips alternate. Change points, outerRadius and innerRadius for different star shapes.
