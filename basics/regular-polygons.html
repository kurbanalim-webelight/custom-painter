What is a regular polygon?

A regular polygon is a shape where all sides are equal and all corners are at the same distance from the centre. For example: triangle (3 sides), square (4), pentagon (5), hexagon (6), octagon (8). If we place the centre at a point and imagine a circle around it, all the corners of the polygon lie on that circle. So we can find each corner using one simple idea: turn by a fixed angle and place a point on the circle.

Triangle, Pentagon, Hexagon, Octagon

Same idea for all of them: we decide the number of sides (3 = triangle, 5 = pentagon, 6 = hexagon, 8 = octagon). We divide the full circle (360¬∞) by that number to get the angle between one corner and the next. Then we put one point at each of those angles on a circle of given radius, and join them with lines to get the polygon.

  • Triangle ‚Äî 3 sides ‚Üí angle step = 360¬∞ √∑ 3 = 120¬∞
  • Pentagon ‚Äî 5 sides ‚Üí angle step = 360¬∞ √∑ 5 = 72¬∞
  • Hexagon ‚Äî 6 sides ‚Üí angle step = 360¬∞ √∑ 6 = 60¬∞
  • Octagon ‚Äî 8 sides ‚Üí angle step = 360¬∞ √∑ 8 = 45¬∞

So the rule is: angle step = 360¬∞ √∑ number of sides. We use this same formula for any regular polygon.

Using angle math: 360¬∞ √∑ sides

In code we work in radians, not degrees. Full circle = 2œÄ radians (same as 360¬∞). So the angle between two neighbouring corners is:

angleStep = (2 * pi) / sides

For the first corner we start at some angle (e.g. 0 or -œÄ/2 if we want the top point first). Then for corner 0 we use angle 0, for corner 1 we use angleStep, for corner 2 we use 2 * angleStep, and so on. So for corner i: angle = i * angleStep (plus the start angle if you added one).

Finding each point: cos and sin

Once we have the angle for a corner, we need to convert ‚Äúangle on a circle‚Äù into x and y. We use:

Offset(centerX + radius * cos(angle), centerY + radius * sin(angle))

Here, centerX and centerY are the centre of the polygon, and radius is the distance from centre to any corner (the circle on which all corners lie). cos(angle) gives how far the point is in the horizontal direction from the centre, and sin(angle) in the vertical direction. So we get the exact position of that corner. We do this for each corner (0 to sides-1), put them in a Path, and close the path to get the polygon.

üí° In short: For each corner i, angle = i * (2œÄ / sides), then point = Offset(centerX + radius * cos(angle), centerY + radius * sin(angle)). Join the points in order and close the path ‚Äî that‚Äôs your regular polygon.

Example code

Below is a small example: a helper that builds a Path for any regular polygon given centre, radius and number of sides. You can use 3 for triangle, 5 for pentagon, 6 for hexagon, 8 for octagon.

@override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()
      ..style = PaintingStyle.stroke
      ..strokeWidth = 3
      ..color = Colors.blue;

    Path regularPolygonPath({
      required Offset center,
      required double radius,
      required int sides,
      double startAngle = -math.pi / 2, // top point first
    }) {
      final path = Path();
      final angleStep = (2 * math.pi) / sides;
      for (var i = 0; i <= sides; i++) {
        final angle = startAngle + i * angleStep;
        final x = center.dx + radius * math.cos(angle);
        final y = center.dy + radius * math.sin(angle);
        if (i == 0) {
          path.moveTo(x, y);
        } else {
          path.lineTo(x, y);
        }
      }
      path.close();
      return path;
    }

    // Usage: triangle, pentagon, hexagon, octagon
    canvas.drawPath(
      regularPolygonPath(
        center: Offset(size.width / 2, size.height / 2),
        radius: 50,
        sides: 8,
      ),
      paint,
    );
  }
