‚≠ï What is Radius?

Radius in Flutter is used to define how much a corner is rounded. You will see it in RRect (rounded rectangles), BorderRadius, and in path methods like arcToPoint(..., radius: Radius.circular(20)).

In simple words: Think of it like the ‚Äúroundness‚Äù of a corner ‚Äî bigger radius means more round, smaller means sharper. You can use the same radius for all corners or different ones for each corner.

Two common constructors:

  • Radius.circular(r) ‚Äî same radius in both X and Y. Gives a circular arc at the corner.
  • Radius.elliptical(x, y) ‚Äî different horizontal (x) and vertical (y) radii. Gives an elliptical arc.
üí° Tip: For RRect and paths, you pass Radius.circular(20) or Radius.elliptical(30, 10) to control how rounded the corners are.
@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()..color = Colors.blue..style = PaintingStyle.stroke..strokeWidth = 3;

  // Same radius on all corners (circular)
  final rect1 = Rect.fromLTWH(40, 40, 120, 80);
  final rrect1 = RRect.fromRectAndRadius(rect1, const Radius.circular(20));
  canvas.drawRRect(rrect1, paint);

  // Elliptical radius (wider horizontally than vertically)
  final rect2 = Rect.fromLTWH(200, 40, 120, 80);
  final rrect2 = RRect.fromRectAndCorners(
    rect2,
    topLeft: const Radius.elliptical(35, 15),
    topRight: const Radius.elliptical(35, 15),
    bottomLeft: const Radius.elliptical(35, 15),
    bottomRight: const Radius.elliptical(35, 15),
  );
  canvas.drawRRect(rrect2, paint);
}
