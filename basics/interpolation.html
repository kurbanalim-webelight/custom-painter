Explanation

Interpolation means computing a value ‚Äúbetween‚Äù two values. In CustomPainter this is used for smooth transitions: animating position, color, size, or opacity over time, or blending between two states. The most common form is linear interpolation (lerp).

Given a start value A, an end value B, and a parameter t in the range 0 to 1, the linearly interpolated value is A + (B ‚àí A) √ó t. When t = 0 you get A; when t = 1 you get B; in between you get a smooth blend. Flutter provides lerpDouble, Offset.lerp, Color.lerp, Rect.lerp, and similar so you don‚Äôt have to implement this by hand.

Use cases: animated painters (t driven by animation controller), morphing shapes, fading colors, moving elements along a path, and any ‚Äúfrom state A to state B‚Äù visual change.

Formulas

Linear interpolation formula

For two values a and b, and parameter t in [0, 1]:

lerp(a, b, t) = a + (b ‚àí a) √ó t

So t = 0 gives a, t = 1 gives b, and values in between give a straight-line blend. For positions, use Offset.lerp(offsetA, offsetB, t); for doubles, lerpDouble(a, b, t) in Flutter‚Äôs ui or dart:ui.

Visual Demo

class _InterpolationDemoScreenState extends State<InterpolationDemoScreen>
    with SingleTickerProviderStateMixin {
  late AnimationController _controller;
  late Animation<double> _animation;

  @override
  void initState() {
    super.initState();
    _controller = AnimationController(
      vsync: this,
      duration: const Duration(milliseconds: 2500),
    )..repeat(reverse: true);
    _animation = CurvedAnimation(parent: _controller, curve: Curves.easeInOut);
  }

  @override
  void dispose() { _controller.dispose(); super.dispose(); }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Interpolation Demo'), centerTitle: true),
      body: AnimatedBuilder(
        animation: _animation,
        builder: (context, _) => CustomPaint(
          painter: InterpolationDemoPainter(t: _animation.value),
          size: Size.infinite,
        ),
      ),
    );
  }
}

class InterpolationDemoPainter extends CustomPainter {
  InterpolationDemoPainter({required this.t});
  /// Interpolation parameter: 0 ‚Üí 1. value = start + (end - start) * t
  final double t;

  @override
  void paint(Canvas canvas, Size size) {
    const margin = 60.0;
    final startX = margin;
    final endX = size.width - margin;
    final centerY = size.height / 2;

    // ---- Track (start to end line) ----
    final trackPaint = Paint()
      ..color = Colors.grey.withValues(alpha: 0.4)
      ..strokeWidth = 4
      ..style = PaintingStyle.stroke;
    canvas.drawLine(Offset(startX, centerY), Offset(endX, centerY), trackPaint);

    // ---- Position: linear interpolation ----
    // x = startX + (endX - startX) * t
    final x = startX + (endX - startX) * t;

    // ---- Radius: interpolate from small to large via t ----
    const minRadius = 20.0;
    const maxRadius = 50.0;
    final radius = minRadius + (maxRadius - minRadius) * t;

    // ---- Opacity: interpolate for visual effect ----
    final opacity = 0.5 + 0.5 * t;
    final circlePaint = Paint()
      ..color = Colors.green.withValues(alpha: opacity)
      ..style = PaintingStyle.fill;
    final strokePaint = Paint()..color = Colors.green..style = PaintingStyle.stroke..strokeWidth = 2;
    canvas.drawCircle(Offset(x, centerY), radius, circlePaint);
    canvas.drawCircle(Offset(x, centerY), radius, strokePaint);

    // ---- Label: t and formula ----
    final textStyle = TextStyle(color: Colors.grey[800], fontSize: 16, fontWeight: FontWeight.w500);
    final paragraph = TextPainter(
      text: TextSpan(style: textStyle, children: [
        const TextSpan(text: 't = '),
        TextSpan(text: t.toStringAsFixed(2), style: TextStyle(color: Colors.green[700])),
        const TextSpan(text: '\nposition = start + (end ‚àí start) √ó t\nradius & opacity also lerped'),
      ]),
      textDirection: TextDirection.ltr,
    )..layout(maxWidth: size.width - 32);
    paragraph.paint(canvas, const Offset(16, 24));
  }

  @override
  bool shouldRepaint(covariant InterpolationDemoPainter oldDelegate) => oldDelegate.t != t;
}
