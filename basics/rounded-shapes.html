Overview

Rounded shapes in CustomPainter come in two main ways: RRect (rounded rectangles drawn with drawRRect) and custom paths where you add rounded corners using arcTo or arcToPoint. Use RRect when you have a rectangle (or rounded rect). Use a Path with arcs when you need rounded corners on a custom outline (e.g. a polygon or an irregular shape).

Using RRect with different corner radii

drawRRect(RRect rrect, Paint paint)

An RRect is a rounded rectangle: a rectangle whose corners are replaced by quarter-circles (or ellipses). You can give the same radius to all four corners (e.g. RRect.fromRectAndRadius(rect, Radius.circular(20))) or different radii per corner (e.g. RRect.fromRectAndCorners, RRect.fromLTRBAndCorners). Then you draw it with canvas.drawRRect(rrect, paint). Ideal for cards, buttons, and any UI that needs a simple rounded box.

Custom rounded paths with arcTo

arcToPoint — Rounded corners on a path

For custom shapes (polygons, tabs, blobs) you build a Path and add rounded corners using arcToPoint: it draws an arc from the current point to a target point with a given corner radius. Use it between lineTo segments to get smooth rounded corners instead of sharp angles.
