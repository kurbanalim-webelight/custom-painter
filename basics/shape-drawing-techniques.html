Overview

This page summarises how to draw shapes with different looks: filled (solid colour), outlined (stroke only), gradient filled (shader), and multiple shapes (several draws, different Paint objects, or combined paths). Each technique uses the Paint object and the Canvas draw APIs explained elsewhere; below we link to the relevant topics.

Filled shapes

Solid fill

To draw a shape with a solid colour inside, set Paint.color to the desired colour and Paint.style = PaintingStyle.fill. Then use any canvas draw method (drawRect, drawCircle, drawPath, etc.). The interior of the shape is filled with that colour.

final paint = Paint()
  ..color = Colors.blue
  ..style = PaintingStyle.fill;
canvas.drawRect(rect, paint);

Outlined shapes

Stroke only

To draw only the outline (border) of a shape, set Paint.style = PaintingStyle.stroke and Paint.strokeWidth to the line thickness. The same draw methods are used; only the outline is drawn, not the interior.

final paint = Paint()
  ..color = Colors.blue
  ..style = PaintingStyle.stroke
  ..strokeWidth = 2.0;
canvas.drawRect(rect, paint);

Gradient filled shapes

Fill with a gradient

Instead of a solid colour, set Paint.shader to a shader created from a LinearGradient, RadialGradient, or SweepGradient via createShader(rect). The shape is then filled with that gradient. When shader is set, color is ignored.

final paint = Paint()
  ..shader = LinearGradient(
    colors: [Colors.blue, Colors.purple],
  ).createShader(rect);
canvas.drawRect(rect, paint);

Multiple shapes

To draw several shapes in one paint() call:

  • Draw multiple shapes by calling draw methods multiple times. Use the same Canvas and call drawRect, drawCircle, drawPath, etc., as many times as you need. Each call draws one shape. See the Canvas and Size page for the drawing API, and Primitive Shapes / Complex Shapes for the full list of draw methods.
  • Use different Paint objects for different styles. Create one Paint for filled blue, another for stroked red, another with a gradient, and pass the right one to each draw call. See What is Paint? for all Paint properties.
  • Combine shapes using paths. Build one Path from multiple segments (lines, curves, arcs) and draw it once with drawPath. Useful for custom icons or a single outline made of many parts. See the Path page and Path Building Methods.
