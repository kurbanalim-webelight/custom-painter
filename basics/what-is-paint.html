üìö What is Paint?

Paint is a class in Flutter (from dart:ui) that describes the style and configuration for drawing operations. It controls HOW things are drawn - their color, thickness, effects, and appearance.

Think of Paint as:

  • Your paintbrush configuration
  • The settings on your drawing tool
  • Style instructions for the canvas
  • The "look and feel" of what you draw
üí° Key Concept: Paint is separate from Canvas. Canvas defines WHAT to draw (shapes, paths), Paint defines HOW to draw it (color, style, effects).

üé® Paint Properties (Complete A-Z Reference)

color

Type: Color color

Definition: The color to use when painting. This is the most fundamental Paint property.

What Happens:

  • Sets the color of filled shapes or stroked lines
  • Applies to everything drawn with this Paint object
  • Includes alpha channel for transparency
  • If a shader is set, color is ignored (shader takes precedence)
@override
void paint(Canvas canvas, Size size) {
  // Solid red circle
  final redPaint = Paint()..color = Colors.red;
  canvas.drawCircle(Offset(50, size.height / 2), 30, redPaint);

  // Blue with 50% opacity
  final semiTransparentBluePaint = Paint()
    ..color = Colors.blue.withValues(alpha: 0.5);
  canvas.drawCircle(
    Offset(100, size.height / 2),
    30,
    semiTransparentBluePaint,
  );

  // Custom color using ARGB
  final customPaint = Paint()
    ..color = Color.fromARGB(255, 100, 200, 150); // RGB values
  canvas.drawCircle(Offset(150, size.height / 2), 30, customPaint);

  // Hex color
  final hexColorPaint = Paint()..color = Color(0xFF42A5F5);
  canvas.drawCircle(Offset(200, size.height / 2), 30, hexColorPaint);
}

isAntiAlias

Type: bool isAntiAlias

Definition: Controls whether edges are smoothed (anti-aliased) or appear jagged (aliased). Anti-aliasing makes edges look smoother by blending pixels.

What Happens:

  • true (default) - Smooth, professional-looking edges
  • false - Sharp, pixelated edges (faster performance)
  • Affects lines, curves, and shape edges
  • Small performance cost when enabled
@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()
    ..color = Colors.red
    ..style = PaintingStyle.stroke
    ..strokeWidth = 10
    ..isAntiAlias = false;

  // Diagonal line
  canvas.drawLine(
    const Offset(20, 20),
    Offset(size.width - 20, size.height - 20),
    paint,
  );

  // Circle
  canvas.drawCircle(Offset(size.width / 2, size.height / 2), 60, paint);

  final antiAliasPaint = Paint()
    ..color = Colors.green
    ..style = PaintingStyle.stroke
    ..strokeWidth = 10
    ..isAntiAlias = true;

  // Diagonal line
  canvas.drawLine(
    Offset(size.width * 0.5, size.height * 1.5),
    Offset(size.width * 1.5, size.height * 1.5),
    antiAliasPaint,
  );

  // Circle
  canvas.drawCircle(
    Offset(size.width * 0.5, size.height * 1.5),
    60,
    antiAliasPaint,
  );
}

The top circle is aliased (pixelated), the bottom circle is anti-aliased (smooth).

invertColors

Type: bool invertColors

Definition: When true, inverts the colors of everything painted with this Paint. Black becomes white, red becomes cyan, etc.

What Happens:

  • RGB values are inverted: newColor = 255 - oldColor
  • Creates negative/inverted image effect
  • Useful for dark mode adjustments
  • Alpha channel is NOT inverted
@override
void paint(Canvas canvas, Size size) {
  // Normal paint
  final normalPaint = Paint()
    ..color = Colors.blue
    ..invertColors = false;

  // Inverted paint
  final invertedPaint = Paint()
    ..color = Colors.blue
    ..invertColors = true;

  final centerY = size.height / 2;

  // Normal circle
  canvas.drawCircle(Offset(size.width * 0.35, centerY), 60, normalPaint);

  // Inverted circle
  canvas.drawCircle(Offset(size.width * 0.65, centerY), 60, invertedPaint);
}

Both of the circles are using blue color, but the right one is inverted.

shader

Type: Shader? shader

Definition: Applies a gradient or pattern fill instead of a solid color. Shaders provide advanced color effects like linear gradients, radial gradients, and sweep gradients.

What Happens:

  • Replaces solid color with gradient or pattern
  • When shader is set, the color property is ignored
  • Shaders are created from Gradient classes
  • Can create complex, beautiful effects

Shader Types (from Gradients):

  • LinearGradient - Color transition along a line
  • RadialGradient - Color transition from center outward
  • SweepGradient - Color transition around a circle
@override
void paint(Canvas canvas, Size size) {
  // Linear gradient (left to right)
  final linearGradient = LinearGradient(
    colors: [Colors.blue, Colors.red],
    begin: Alignment.centerLeft,
    end: Alignment.centerRight,
  );

  final rect1 = Rect.fromLTWH(10, 10, 100, 100);
  canvas.drawRect(
    rect1,
    Paint()..shader = linearGradient.createShader(rect1),
  );

  // Radial gradient (center outward)
  final radialGradient = RadialGradient(
    colors: [Colors.yellow, Colors.orange, Colors.red],
    stops: [0.0, 0.5, 1.0],
  );

  final rect2 = Rect.fromLTWH(120, 10, 100, 100);
  canvas.drawCircle(
    Offset(170, 60),
    50,
    Paint()..shader = radialGradient.createShader(rect2),
  );

  // Sweep gradient (around circle)
  final sweepGradient = SweepGradient(
    colors: [
      Colors.red,
      Colors.yellow,
      Colors.green,
      Colors.cyan,
      Colors.blue,
      Colors.magenta,
      Colors.red,
    ],
  );

  final rect3 = Rect.fromLTWH(230, 10, 100, 100);
  canvas.drawCircle(
    Offset(280, 60),
    50,
    Paint()..shader = sweepGradient.createShader(rect3),
  );

  // Multi-stop gradient
  final complexGradient = LinearGradient(
    colors: [
      Colors.purple,
      Colors.blue,
      Colors.green,
      Colors.yellow,
      Colors.red,
    ],
    stops: [0.0, 0.25, 0.5, 0.75, 1.0],
  );

  final rect4 = Rect.fromLTWH(10, 120, 320, 60);
  canvas.drawRect(
    rect4,
    Paint()..shader = complexGradient.createShader(rect4),
  );
}

strokeCap

Type: StrokeCap strokeCap

Definition: Controls how the ends of lines (strokes) are drawn. Determines the shape of line terminals.

What Happens:

  • Affects the appearance of line endings
  • Only applies when style = PaintingStyle.stroke
  • Can make lines look rounded, square, or flat
  • Affects perceived line length

StrokeCap Values:

  • StrokeCap.butt - Flat end, exactly at endpoint (default)
  • StrokeCap.round - Rounded semicircular end
  • StrokeCap.square - Square end, extends beyond endpoint by strokeWidth/2
@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()
    ..color = Colors.blue
    ..strokeWidth = 20
    ..style = PaintingStyle.stroke;

  // Butt cap (flat, exact endpoint)
  canvas.drawLine(
    Offset(50, 50),
    Offset(200, 50),
    paint..strokeCap = StrokeCap.butt,
  );

  // Round cap (rounded ends)
  canvas.drawLine(
    Offset(50, 100),
    Offset(200, 100),
    paint..strokeCap = StrokeCap.round,
  );

  // Square cap (extends beyond endpoint)
  canvas.drawLine(
    Offset(50, 150),
    Offset(200, 150),
    paint..strokeCap = StrokeCap.square,
  );

  // Practical example: Round caps for progress bars
  canvas.drawArc(
    Rect.fromCircle(center: Offset(150, 250), radius: 50),
    -pi / 2, // Start at top
    pi, // Half circle
    false, // Don't use center
    Paint()
      ..color = Colors.green
      ..strokeWidth = 10
      ..style = PaintingStyle.stroke
      ..strokeCap = StrokeCap.round, // Rounded ends look professional
  );
}

strokeJoin

Type: StrokeJoin strokeJoin

Definition: Controls how corners (joins) between connected line segments are drawn. Determines the shape at path vertices.

What Happens:

  • Affects the appearance of corners in paths and polygons
  • Only applies when style = PaintingStyle.stroke
  • Can make corners rounded, sharp, or beveled
  • Affects how thick corners appear

StrokeJoin Values:

  • StrokeJoin.miter - Sharp pointed corner (default), can extend far
  • StrokeJoin.round - Rounded corner
  • StrokeJoin.bevel - Flat diagonal corner
@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()
    ..color = Colors.blue
    ..strokeWidth = 15
    ..style = PaintingStyle.stroke;

  // Miter join (sharp corners)
  final miterPath = Path()
    ..moveTo(50, 50)
    ..lineTo(100, 100)
    ..lineTo(150, 50)
    ..lineTo(200, 100);

  canvas.drawPath(
    miterPath,
    paint..strokeJoin = StrokeJoin.miter,
  );

  // Round join (rounded corners)
  final roundPath = Path()
    ..moveTo(50, 150)
    ..lineTo(100, 200)
    ..lineTo(150, 150)
    ..lineTo(200, 200);

  canvas.drawPath(
    roundPath,
    paint..strokeJoin = StrokeJoin.round,
  );

  // Bevel join (flat corners)
  final bevelPath = Path()
    ..moveTo(50, 250)
    ..lineTo(100, 300)
    ..lineTo(150, 250)
    ..lineTo(200, 300);

  canvas.drawPath(
    bevelPath,
    paint..strokeJoin = StrokeJoin.bevel,
  );
}

strokeWidth

Type: double strokeWidth

Definition: The thickness of lines and shape outlines when using stroke style. Measured in logical pixels.

What Happens:

  • Controls how thick lines and outlines appear
  • Only visible when style = PaintingStyle.stroke
  • Grows outward from the path center
  • Affected by canvas scaling
  • Default is 0.0 (hairline width)
@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.stroke;

  // Very thin line (1 pixel)
  canvas.drawLine(
    Offset(50, 20),
    Offset(250, 20),
    paint..strokeWidth = 1,
  );

  // Medium line (5 pixels)
  canvas.drawLine(
    Offset(50, 50),
    Offset(250, 50),
    paint..strokeWidth = 5,
  );

  // Thick line (10 pixels)
  canvas.drawLine(
    Offset(50, 90),
    Offset(250, 90),
    paint..strokeWidth = 10,
  );

  // Very thick line (20 pixels)
  canvas.drawLine(
    Offset(50, 140),
    Offset(250, 140),
    paint..strokeWidth = 20,
  );

  // Circles with different stroke widths
  canvas.drawCircle(
    Offset(100, 280),
    30,
    paint..strokeWidth = 2,
  );

  canvas.drawCircle(
    Offset(180, 280),
    30,
    paint..strokeWidth = 8,
  );

  canvas.drawCircle(
    Offset(260, 280),
    30,
    paint..strokeWidth = 15,
  );
}

style

Type: PaintingStyle style

Definition: Determines whether shapes are filled or only outlined (stroked). This is one of the most important Paint properties.

What Happens:

  • PaintingStyle.fill (default) - Fills the interior of shapes
  • PaintingStyle.stroke - Draws only the outline/border
  • Affects all shape drawing operations
  • Changes how strokeWidth, strokeCap, strokeJoin are applied
@override
void paint(Canvas canvas, Size size) {
  // Filled circle (default)
  canvas.drawCircle(
    Offset(70, 70),
    40,
    Paint()
      ..color = Colors.blue
      ..style = PaintingStyle.fill, // Default
  );

  // Stroked circle (outline only)
  canvas.drawCircle(
    Offset(180, 70),
    40,
    Paint()
      ..color = Colors.blue
      ..strokeWidth = 3
      ..style = PaintingStyle.stroke,
  );

  // Filled rectangle
  canvas.drawRect(
    Rect.fromLTWH(30, 140, 80, 60),
    Paint()
      ..color = Colors.red
      ..style = PaintingStyle.fill,
  );

  // Stroked rectangle
  canvas.drawRect(
    Rect.fromLTWH(140, 140, 80, 60),
    Paint()
      ..color = Colors.red
      ..strokeWidth = 4
      ..style = PaintingStyle.stroke,
  );

  // Practical example: Button with border
  final buttonRect = Rect.fromLTWH(50, 350, 150, 50);

  // Fill background
  canvas.drawRRect(
    RRect.fromRectAndRadius(buttonRect, Radius.circular(25)),
    Paint()
      ..color = Colors.blue.withOpacity(0.3)
      ..style = PaintingStyle.fill,
  );

  // Draw border
  canvas.drawRRect(
    RRect.fromRectAndRadius(buttonRect, Radius.circular(25)),
    Paint()
      ..color = Colors.blue
      ..strokeWidth = 2
      ..style = PaintingStyle.stroke,
  );
}

üéØ Complete Paint Example: Combining Properties

Here's a comprehensive example showing multiple Paint properties working together:

class ComprehensivePaintDemo extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // Example 1: Glowing gradient circle
    final glowPaint = Paint()
      ..shader = RadialGradient(
        colors: [Colors.blue, Colors.purple],
      ).createShader(Rect.fromCircle(
        center: Offset(80, 80),
        radius: 50,
      ))
      ..maskFilter = MaskFilter.blur(BlurStyle.normal, 10)
      ..style = PaintingStyle.fill
      ..isAntiAlias = true;

    canvas.drawCircle(Offset(80, 80), 50, glowPaint);

    // Example 2: Multi-gradient rectangle with shadow
    canvas.save();
    canvas.translate(80, 200);

    // Shadow layer
    final shadowPaint = Paint()
      ..color = Colors.black.withOpacity(0.3)
      ..maskFilter = MaskFilter.blur(BlurStyle.normal, 8);

    canvas.drawRRect(
      RRect.fromRectAndRadius(
        Rect.fromLTWH(5, 5, 120, 80),
        Radius.circular(15),
      ),
      shadowPaint,
    );

    // Main shape with gradient
    final gradientPaint = Paint()
      ..shader = LinearGradient(
        colors: [Colors.pink, Colors.purple, Colors.blue],
        begin: Alignment.topLeft,
        end: Alignment.bottomRight,
      ).createShader(Rect.fromLTWH(0, 0, 120, 80))
      ..style = PaintingStyle.fill
      ..isAntiAlias = true;

    canvas.drawRRect(
      RRect.fromRectAndRadius(
        Rect.fromLTWH(0, 0, 120, 80),
        Radius.circular(15),
      ),
      gradientPaint,
    );

    // Border
    final borderStroke = Paint()
      ..color = Colors.white
      ..strokeWidth = 2
      ..style = PaintingStyle.stroke
      ..isAntiAlias = true;

    canvas.drawRRect(
      RRect.fromRectAndRadius(
        Rect.fromLTWH(0, 0, 120, 80),
        Radius.circular(15),
      ),
      borderStroke,
    );

    canvas.restore();
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}
