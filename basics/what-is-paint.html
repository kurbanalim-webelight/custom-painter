üìö What is Paint?

Paint is a class in Flutter (from dart:ui) that describes the style and configuration for drawing operations. It controls HOW things are drawn - their color, thickness, effects, and appearance.

Think of Paint as:

  • Your paintbrush configuration
  • The settings on your drawing tool
  • Style instructions for the canvas
  • The "look and feel" of what you draw
üí° Key Concept: Paint is separate from Canvas. Canvas defines WHAT to draw (shapes, paths), Paint defines HOW to draw it (color, style, effects).

üé® Paint Properties (Complete A-Z Reference)

color

Type: Color color

Definition: The color to use when painting. This is the most fundamental Paint property.

What Happens:

  • Sets the color of filled shapes or stroked lines
  • Applies to everything drawn with this Paint object
  • Includes alpha channel for transparency
  • If a shader is set, color is ignored (shader takes precedence)
@override
void paint(Canvas canvas, Size size) {
  // Solid red circle
  final redPaint = Paint()..color = Colors.red;
  canvas.drawCircle(Offset(50, size.height / 2), 30, redPaint);

  // Blue with 50% opacity
  final semiTransparentBluePaint = Paint()
    ..color = Colors.blue.withValues(alpha: 0.5);
  canvas.drawCircle(
    Offset(100, size.height / 2),
    30,
    semiTransparentBluePaint,
  );

  // Custom color using ARGB
  final customPaint = Paint()
    ..color = Color.fromARGB(255, 100, 200, 150); // RGB values
  canvas.drawCircle(Offset(150, size.height / 2), 30, customPaint);

  // Hex color
  final hexColorPaint = Paint()..color = Color(0xFF42A5F5);
  canvas.drawCircle(Offset(200, size.height / 2), 30, hexColorPaint);
}

isAntiAlias

Type: bool isAntiAlias

Definition: Controls whether edges are smoothed (anti-aliased) or appear jagged (aliased). Anti-aliasing makes edges look smoother by blending pixels.

What Happens:

  • true (default) - Smooth, professional-looking edges
  • false - Sharp, pixelated edges (faster performance)
  • Affects lines, curves, and shape edges
  • Small performance cost when enabled
@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()
    ..color = Colors.red
    ..style = PaintingStyle.stroke
    ..strokeWidth = 10
    ..isAntiAlias = false;

  // Diagonal line
  canvas.drawLine(
    const Offset(20, 20),
    Offset(size.width - 20, size.height - 20),
    paint,
  );

  // Circle
  canvas.drawCircle(Offset(size.width / 2, size.height / 2), 60, paint);

  final antiAliasPaint = Paint()
    ..color = Colors.green
    ..style = PaintingStyle.stroke
    ..strokeWidth = 10
    ..isAntiAlias = true;

  // Diagonal line
  canvas.drawLine(
    Offset(size.width * 0.5, size.height * 1.5),
    Offset(size.width * 1.5, size.height * 1.5),
    antiAliasPaint,
  );

  // Circle
  canvas.drawCircle(
    Offset(size.width * 0.5, size.height * 1.5),
    60,
    antiAliasPaint,
  );
}

The top circle is aliased (pixelated), the bottom circle is anti-aliased (smooth).

invertColors

Type: bool invertColors

Definition: When true, inverts the colors of everything painted with this Paint. Black becomes white, red becomes cyan, etc.

What Happens:

  • RGB values are inverted: newColor = 255 - oldColor
  • Creates negative/inverted image effect
  • Useful for dark mode adjustments
  • Alpha channel is NOT inverted
@override
void paint(Canvas canvas, Size size) {
  // Normal paint
  final normalPaint = Paint()
    ..color = Colors.blue
    ..invertColors = false;

  // Inverted paint
  final invertedPaint = Paint()
    ..color = Colors.blue
    ..invertColors = true;

  final centerY = size.height / 2;

  // Normal circle
  canvas.drawCircle(Offset(size.width * 0.35, centerY), 60, normalPaint);

  // Inverted circle
  canvas.drawCircle(Offset(size.width * 0.65, centerY), 60, invertedPaint);
}

Both of the circles are using blue color, but the right one is inverted.

shader

Type: Shader? shader

Definition: Applies a gradient or pattern fill instead of a solid color. Shaders provide advanced color effects like linear gradients, radial gradients, and sweep gradients.

What Happens:

  • Replaces solid color with gradient or pattern
  • When shader is set, the color property is ignored
  • Shaders are created from Gradient classes
  • Can create complex, beautiful effects

Shader Types (from Gradients):

  • LinearGradient - Color transition along a line
  • RadialGradient - Color transition from center outward
  • SweepGradient - Color transition around a circle
@override
void paint(Canvas canvas, Size size) {
  final centerY = size.height / 2;
  const radius = 60.0;

  // 1Ô∏è‚É£ Linear Gradient Circle
  final linearPaint = Paint()
    ..shader =
      const LinearGradient(
        colors: [Colors.blue, Colors.purple, Colors.pink],
      ).createShader(
        Rect.fromCircle(
          center: Offset(size.width * 0.25, centerY),
          radius: radius,
        ),
      );

  canvas.drawCircle(Offset(size.width * 0.25, centerY), radius, linearPaint);

  // 2Ô∏è‚É£ Radial Gradient Circle (Glow effect)
  final radialPaint = Paint()
    ..shader =
      RadialGradient(
        colors: [Colors.yellow, Colors.orange, Colors.red],
      ).createShader(
        Rect.fromCircle(
          center: Offset(size.width * 0.5, centerY),
          radius: radius,
        ),
      );

  canvas.drawCircle(Offset(size.width * 0.5, centerY), radius, radialPaint);

  // 3Ô∏è‚É£ Sweep Gradient Circle (Progress / Radar)
  final sweepPaint = Paint()
    ..shader =
      const SweepGradient(
        colors: [Colors.green, Colors.blue, Colors.purple, Colors.green],
      ).createShader(
        Rect.fromCircle(
          center: Offset(size.width * 0.75, centerY),
          radius: radius,
        ),
      );

  canvas.drawCircle(Offset(size.width * 0.75, centerY), radius, sweepPaint);
}

The three circles are using different shaders.

strokeCap

Type: StrokeCap strokeCap

Definition: Controls how the ends of lines (strokes) are drawn. Determines the shape of line terminals.

What Happens:

  • Affects the appearance of line endings
  • Only applies when style = PaintingStyle.stroke
  • Can make lines look rounded, square, or flat
  • Affects perceived line length

StrokeCap Values:

  • StrokeCap.butt - Flat end, exactly at endpoint (default)
  • StrokeCap.round - Rounded semicircular end
  • StrokeCap.square - Square end, extends beyond endpoint by strokeWidth/2
@override
void paint(Canvas canvas, Size size) {
  final strokeWidth = 16.0;

  // Center the three lines vertically with spacing in between

  // Calculate vertical center and spacing
  final centerY = size.height / 2;
  final spacing = 40.0; // space between lines

  // y coordinates for each line
  final y1 = centerY - spacing;
  final y2 = centerY;
  final y3 = centerY + spacing;

  // 1Ô∏è‚É£ StrokeCap.butt (default)
  final buttPaint = Paint()
    ..color = Colors.red
    ..strokeWidth = strokeWidth
    ..style = PaintingStyle.stroke
    ..strokeCap = StrokeCap.butt;

  canvas.drawLine(Offset(40, y1), Offset(size.width - 40, y1), buttPaint);

  // 2Ô∏è‚É£ StrokeCap.round
  final roundPaint = Paint()
    ..color = Colors.green
    ..strokeWidth = strokeWidth
    ..style = PaintingStyle.stroke
    ..strokeCap = StrokeCap.round;

  canvas.drawLine(Offset(40, y2), Offset(size.width - 40, y2), roundPaint);

  // 3Ô∏è‚É£ StrokeCap.square
  final squarePaint = Paint()
    ..color = Colors.blue
    ..strokeWidth = strokeWidth
    ..style = PaintingStyle.stroke
    ..strokeCap = StrokeCap.square;

  canvas.drawLine(Offset(40, y3), Offset(size.width - 40, y3), squarePaint);
}

The three lines are using different stroke caps. The butt cap is flat, the round cap is rounded, and the square cap extends beyond the endpoint by half of the stroke width.

strokeJoin

Type: StrokeJoin strokeJoin

Definition: Controls how corners (joins) between connected line segments are drawn. Determines the shape at path vertices.

What Happens:

  • Affects the appearance of corners in paths and polygons
  • Only applies when style = PaintingStyle.stroke
  • Can make corners rounded, sharp, or beveled
  • Affects how thick corners appear

StrokeJoin Values:

  • StrokeJoin.miter - Sharp pointed corner (default), can extend far
  • StrokeJoin.round - Rounded corner
  • StrokeJoin.bevel - Flat diagonal corner
@override
void paint(Canvas canvas, Size size) {
  const strokeWidth = 26.0;

  final centerX = size.width / 2;
  final spacing = 100.0;

  // Helper function to draw a V shape
  void drawVShape(Paint paint, double offsetY) {
    final path = Path()
      ..moveTo(centerX - 80, offsetY)
      ..lineTo(centerX, offsetY + 80)
      ..lineTo(centerX + 80, offsetY);

    canvas.drawPath(path, paint);
  }

  // 1Ô∏è‚É£ StrokeJoin.miter (sharp corner)
  final miterPaint = Paint()
    ..color = Colors.red
    ..style = PaintingStyle.stroke
    ..strokeWidth = strokeWidth
    ..strokeJoin = StrokeJoin.miter;

  drawVShape(miterPaint, spacing);

  // 2Ô∏è‚É£ StrokeJoin.round (rounded corner)
  final roundPaint = Paint()
    ..color = Colors.green
    ..style = PaintingStyle.stroke
    ..strokeWidth = strokeWidth
    ..strokeJoin = StrokeJoin.round;

  drawVShape(roundPaint, spacing * 2);

  // 3Ô∏è‚É£ StrokeJoin.bevel (flat corner)
  final bevelPaint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.stroke
    ..strokeWidth = strokeWidth
    ..strokeJoin = StrokeJoin.bevel;

  drawVShape(bevelPaint, spacing * 3);
}

The three V shapes are using different stroke joins. The miter join is sharp, the round join is rounded, and the bevel join is flat.

strokeWidth

Type: double strokeWidth

Definition: The thickness of lines and shape outlines when using stroke style. Measured in logical pixels.

What Happens:

  • Controls how thick lines and outlines appear
  • Only visible when style = PaintingStyle.stroke
  • Grows outward from the path center
  • Affected by canvas scaling
  • Default is 0.0 (hairline width)
@override
void paint(Canvas canvas, Size size) {
  final centerY = size.height / 2;
  const spacing = 40.0;

  // Thin stroke
  final thinPaint = Paint()
    ..color = Colors.red
    ..style = PaintingStyle.stroke
    ..strokeWidth = 4;

  canvas.drawLine(
    Offset(40, centerY - spacing),
    Offset(size.width - 40, centerY - spacing),
    thinPaint,
  );

  // Medium stroke
  final mediumPaint = Paint()
    ..color = Colors.green
    ..style = PaintingStyle.stroke
    ..strokeWidth = 10;

  canvas.drawLine(
    Offset(40, centerY),
    Offset(size.width - 40, centerY),
    mediumPaint,
  );

  // Thick stroke
  final thickPaint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.stroke
    ..strokeWidth = 20;

  canvas.drawLine(
    Offset(40, centerY + spacing),
    Offset(size.width - 40, centerY + spacing),
    thickPaint,
  );
}

The two circles are using different stroke widths. The filled circle is 50 pixels wide, and the stroked circle is 8 pixels wide.

style

Type: PaintingStyle style

Definition: Determines whether shapes are filled or only outlined (stroked). This is one of the most important Paint properties.

What Happens:

  • PaintingStyle.fill (default) - Fills the interior of shapes
  • PaintingStyle.stroke - Draws only the outline/border
  • Affects all shape drawing operations
  • Changes how strokeWidth, strokeCap, strokeJoin are applied
@override
void paint(Canvas canvas, Size size) {
  final centerY = size.height / 2;
  const radius = 50.0;

  // 1Ô∏è‚É£ Filled circle
  final fillPaint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.fill;

  canvas.drawCircle(Offset(size.width * 0.35, centerY), radius, fillPaint);

  // 2Ô∏è‚É£ Stroked circle
  final strokePaint = Paint()
    ..color = Colors.red
    ..style = PaintingStyle.stroke
    ..strokeWidth = 8;

  canvas.drawCircle(Offset(size.width * 0.65, centerY), radius, strokePaint);
}

The filled circle is filled with blue color, and the stroked circle is outlined with red color.

üéØ Complete Paint Example: Combining Properties

Here's a comprehensive example showing multiple Paint properties working together:

@override
void paint(Canvas canvas, Size size) {
  final center = Offset(size.width / 2, size.height / 2);
  const radius = 80.0;

  // üé® Paint used for progress ring
  final paint = Paint()
    // Stroke only (ring, not filled circle)
    ..style = PaintingStyle.stroke
    // Thickness of the ring
    ..strokeWidth = 18
    // Rounded ends (looks premium)
    ..strokeCap = StrokeCap.round
    // Smooth curve joins
    ..strokeJoin = StrokeJoin.round
    // Smooth edges
    ..isAntiAlias = true
    // Gradient progress color
    ..shader = const LinearGradient(
      colors: [Colors.green, Colors.yellow, Colors.orange, Colors.red],
      begin: Alignment.topLeft,
      end: Alignment.bottomRight,
    ).createShader(
      Rect.fromCircle(center: center, radius: radius),
    );

  canvas.drawArc(
    Rect.fromCircle(center: center, radius: radius),
    -math.pi / 2, // Start from top (12 o'clock)
    math.pi * 1.5, // 75% of full circle (2œÄ)
    false,
    paint,
  );

  canvas.drawArc(
    Rect.fromCircle(center: center, radius: radius / 2 + 10),
    -math.pi / 2, // Start from top (12 o'clock)
    math.pi * 1.2, // Partial circle
    false,
    paint,
  );
}

The progress ring is a ring with a gradient color.
