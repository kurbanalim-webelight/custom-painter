Overview

Complex curves are paths built from more than one kind of segment: lines, quadratic Bezier curves, cubic Bezier curves, and arcs. Flutter’s Path lets you mix lineTo, quadraticBezierTo, cubicTo, arcTo, arcToPoint, and their relative variants in a single path. You use Bezier methods for smooth, controllable curves; combining several curve types in one path gives you complex shapes (e.g. custom icons, waves, or organic outlines).

Bezier curves for smooth shapes

Bezier curves are smooth curves defined by control points. Quadratic Bezier uses one control point; cubic uses two. They are the main way to draw smooth, curved shapes that are not simple arcs. Two core methods:

quadraticBezierTo — Absolute quadratic curve

Draws a quadratic Bezier curve from the current point to an end point, with one control point. All coordinates are in absolute (canvas) space. Ideal when you know the exact positions of control and end points.

relativeQuadraticBezierTo — Relative quadratic curve

Same as quadraticBezierTo but the control point and end point are given relative to the current point. Handy for repeated patterns (e.g. waves or scallops) where each segment is offset from the previous one by fixed deltas.

Combining multiple curve types

A single Path can mix straight lines, quadratic curves, cubic curves, and arcs. You call moveTo once to start, then chain lineTo, quadraticBezierTo, cubicTo, arcTo, arcToPoint, etc. Each call continues from the current point. The result is one path that can be stroked or filled in one go — useful for custom shapes that have both straight and curved sections.

Example: path with line, quadratic curve, and cubic curve

Below, the path starts with a line, then a quadratic Bezier (one control point), then a cubic Bezier (two control points). All in one path, drawn with a single drawPath.

@override
void paint(Canvas canvas, Size size) {
  final paint = Paint()
    ..style = PaintingStyle.stroke
    ..strokeWidth = 3
    ..color = Colors.blue;

  final w = size.width;
  final h = size.height;

  // Combine: line → quadratic → cubic
  final path = Path()
    ..moveTo(40, h * 0.5)
    ..lineTo(w * 0.25, h * 0.5)
    ..quadraticBezierTo(w * 0.4, h * 0.2, w * 0.5, h * 0.5)
    ..cubicTo(w * 0.6, h * 0.2, w * 0.75, h * 0.8, w - 40, h * 0.5);

  canvas.drawPath(path, paint);
}

One path built with lineTo, quadraticBezierTo, and cubicTo. The blue curve shows the straight segment, then the quadratic hump, then the cubic wave.

You can add arcTo, arcToPoint, or relativeQuadraticBezierTo in the same way — each call continues from the last point. Order and choice of method define the shape.
