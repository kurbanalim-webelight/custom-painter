Math helpers in Flutter

Flutter provides a few math helpers that are useful in CustomPainter ‚Äî for example to smoothly change a value over time (animations) or to blend between two numbers. The one we cover here is lerpDouble.

lerpDouble

Signature: double? lerpDouble(num? a, num? b, double t)

Purpose: It does linear interpolation between two numbers a and b. The third argument t is a value usually between 0 and 1. When t = 0, you get a; when t = 1, you get b; and for values in between you get a value between a and b.

In simple words: Think of it as ‚Äúmix two numbers.‚Äù For example, if you want to move something from x = 50 to x = 200, you can use lerpDouble(50, 200, t). When t goes from 0 to 1, the result goes from 50 to 200. So it is very handy for animations and smooth transitions.

Formula: a + (b - a) * t. So when t = 0.5 you get the middle value.

üí° Where to find it: lerpDouble is in the dart:ui library. So you can use it as ui.lerpDouble(0, 100, 0.3) if you have import 'dart:ui' as ui;. There is also lerp in the material or rendering library for other types.
import 'dart:ui' as ui;

@override
void paint(Canvas canvas, Size size) {
  // Example: t from 0 to 1 (e.g. from animation)
  const a = 50.0;
  const b = 250.0;
  final t = 0.4; // 40% between a and b
  final x = ui.lerpDouble(a, b, t)!; // 50 + (250-50)*0.4 = 130

  // Draw a circle that moves from left to right as t changes
  final center = Offset(x, size.height / 2);
  canvas.drawCircle(center, 25, Paint()..color = Colors.blue);

  // Optional: draw line from a to b to show the range
  canvas.drawLine(
    Offset(a, size.height / 2),
    Offset(b, size.height / 2),
    Paint()..color = Colors.grey..strokeWidth = 1,
  );
}

As t goes from 0 to 1, the blue circle moves from a to b. Here t = 0.4 so the circle is 40% of the way.
