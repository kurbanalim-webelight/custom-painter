üìö What is RRect?

RRect stands for Rounded Rectangle. It is a class in Flutter (from dart:ui) that describes a rectangle whose corners are rounded. So it is like a normal Rect, but with a Radius (or different radii per corner) so that the corners are drawn as arcs instead of sharp corners.

In simple words: When you want a box with rounded corners ‚Äî like a card or a button ‚Äî you use RRect. You define the same bounds as a Rect, plus how much each corner is rounded.

Where it is used:

  • canvas.drawRRect(rrect, paint) ‚Äî to draw a rounded rectangle on the canvas.
  • canvas.drawDRRect(outer, inner, paint) ‚Äî to draw a ‚Äúhole‚Äù (inner rounded rect cut from an outer one).
  • Path methods like path.addRRect(rrect) to add a rounded rectangle to a path.
üí° Key concept: You often create an RRect from a Rect and a Radius, e.g. RRect.fromRectAndRadius(rect, Radius.circular(20)). For different corners use RRect.fromRectAndCorners(rect, topLeft: ..., topRight: ..., ...).
