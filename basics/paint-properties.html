Overview

This page is a quick guide to how shapes look on the canvas using the Paint object. It focuses on styling ‚Äì colour, borders, smoothness, gradients and advanced effects.

üìå Note: All basic Paint properties are already explained in detail on the What is Paint? page. Here we will just remind you of them and then go a bit deeper into the advanced styling options blendMode and maskFilter.

Basic styling properties (already covered)

These properties are part of everyday drawing and are fully documented in what-is-paint.html:

  • color ‚Äì Shape colour: controls the solid colour of fills and strokes. See full examples on the Paint basics page.
  • style ‚Äì Fill or stroke: chooses between PaintingStyle.fill (filled shapes) and PaintingStyle.stroke (outline only). Explained with diagrams on the Paint page.
  • strokeWidth ‚Äì Line thickness: sets how thick the stroke border is when using PaintingStyle.stroke.
  • strokeCap ‚Äì Line end style: controls how line ends look (round, square, or flat).
  • strokeJoin ‚Äì Corner style: defines how corners between two strokes are drawn (miter, round, bevel).
  • isAntiAlias ‚Äì Smooth edges: decides whether edges are smooth (true) or jagged but a bit faster (false).
  • shader ‚Äì Gradients: replaces solid colour with gradients like linear, radial and sweep. Full gradient examples are in What is Paint?.
üí° Tip: For day‚Äëto‚Äëday UI work, most styling can be done with just color, style, strokeWidth and shader. When you want more advanced control on how shapes mix with the background or glow/blur, move to Advanced Styling below.

‚ú® Advanced Styling

blendMode ‚Äî How shape blends with background

Type: BlendMode blendMode

In simple words: blendMode controls how the colour from your shape mixes with the colour that is already on the canvas (background or other shapes).

Why we use it

Normally, new drawings simply cover what is behind them (default BlendMode.srcOver). With other blend modes we can create lighting, highlight, darken, multiply and overlay effects like in photo‚Äëediting tools.

Common use cases

  • Darkening or tinting a background without changing the original image.
  • Special visual effects like additive light (BlendMode.plus) or multiply shading (BlendMode.multiply).

class BlendModePropertyCP extends CustomPainter {
  final BlendMode blendMode;

  BlendModePropertyCP({required this.blendMode});

  @override
  void paint(Canvas canvas, Size size) {
    final rect = Rect.fromLTWH(0, 0, size.width, size.height);

    final bgPaint = Paint()..color = Colors.blue;

    final srcPaint = Paint()
      ..color = Colors.yellow
      ..blendMode = blendMode;

    canvas.saveLayer(rect, Paint());

    canvas.drawRect(rect, bgPaint);

    canvas.drawCircle(
      size.center(Offset.zero),
      size.shortestSide * 0.3,
      srcPaint,
    );

    canvas.restore();

    canvas.drawRect(
      rect,
      Paint()
        ..style = PaintingStyle.stroke
        ..color = Colors.black26,
    );

    final textPainter = TextPainter(
      text: TextSpan(
        text: blendMode.name,
        style: const TextStyle(
          color: Colors.white,
          fontSize: 12,
          fontWeight: FontWeight.w600,
        ),
      ),
      textDirection: TextDirection.ltr,
    )..layout(maxWidth: size.width - 8);

    textPainter.paint(canvas, const Offset(6, 6));
  }

  @override
  bool shouldRepaint(covariant BlendModePropertyCP oldDelegate) {
    return oldDelegate.blendMode != blendMode;
  }
}

maskFilter ‚Äî Blur / glow around shapes

Type: MaskFilter? maskFilter

In simple words: maskFilter adds a blur to the edges of what you draw. This is useful for soft glows, shadows or fuzzy borders.

Why we use it

Normal shapes have sharp edges. With maskFilter we can soften those edges and create subtle depth, glow or focus effects, without manually drawing multiple layers.

Common use cases

  • Soft glows behind icons or important UI elements.
  • Light blur on shapes to separate them from the background.
  • Highlighting selected areas in charts or custom controls.
@override
void paint(Canvas canvas, Size size) {
  final center = size.center(Offset.zero);

  // Base circle
  final basePaint = Paint()
    ..color = Colors.deepPurple
    ..style = PaintingStyle.fill;
  canvas.drawCircle(center, 40, basePaint);

  // Glow around circle using maskFilter blur
  final glowPaint = Paint()
    ..color = Colors.deepPurple.withValues(alpha: 0.6)
    ..maskFilter = MaskFilter.blur(BlurStyle.normal, 15);

  canvas.drawCircle(center, 60, glowPaint);
}

Adds a soft blur or glow to the edges of shapes. The clear circle is drawn first. Then, the larger blurred circle is drawn behind, creating the appearance of a gentle glow that spreads beyond the edges. Perfect for making elements stand out or adding focus/highlight effects.

The smaller circle is the main shape, and the larger blurred circle acts like a soft glow behind it. By changing blur style and radius, you can tune the effect.

‚úÖ Summary: Use blendMode when you want your drawing to mix colours with what is already on the canvas. Use maskFilter when you want to soften edges and create glow or blur around shapes. Combined with the basic properties from What is Paint?, you can style almost any custom UI you imagine.
