Explanation

Trigonometry is essential for placing and moving points around circles and for periodic motion in CustomPainter. When you draw polygons, stars, arcs, or waves, you often need a point on a circle at a given angle ‚Äî that is exactly what sine and cosine provide.

In Flutter, the canvas uses a coordinate system where x increases to the right and y increases downward. For a circle centered at (cx, cy) with radius r, the point at angle Œ∏ (in radians) is:

  • x = center + radius √ó cos(Œ∏)
  • y = center + radius √ó sin(Œ∏)

Use dart:math for sin, cos, and pi. Angles in the Flutter canvas are typically measured in radians; 0 radians points to the right (3 o'clock), and increasing the angle goes counter-clockwise (e.g. œÄ/2 is up, œÄ is left). Trigonometry in CustomPainter is used for circular layouts (polygons, stars), arcs, and wave-like motion.

Formulas

sin, cos for circular positioning

To place a point on a circle (or to step around a circle for polygons/stars):

x = cx + r √ó cos(Œ∏)   ,    y = cy + r √ó sin(Œ∏)

Here cx, cy is the center, r is the radius, and Œ∏ is the angle in radians.

Radians conversion

Flutter and dart:math use radians, not degrees. Convert degrees to radians with:

radians = degrees √ó œÄ / 180

In Dart: final radians = degrees * math.pi / 180;

Visual Demo

@override
void paint(Canvas canvas, Size size) {
  final centerX = size.width / 2;
  final centerY = size.height / 2;
  const radius = 120.0;
  const numPoints = 8;

  final center = Offset(centerX, centerY);

  // ---- Center point ----
  final centerPaint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.fill;
  canvas.drawCircle(center, 10, centerPaint);

  // ---- Radius line (from center to first point) ----
  // Convert 0¬∞ to radians: degrees * pi / 180
  const angleRad = 0 * pi / 180;
  final radiusEnd = Offset(
    centerX + radius * cos(angleRad),
    centerY + radius * sin(angleRad),
  );
  final radiusPaint = Paint()
    ..color = Colors.grey
    ..strokeWidth = 2
    ..style = PaintingStyle.stroke;
  canvas.drawLine(center, radiusEnd, radiusPaint);

  // ---- Circle outline (optional visual) ----
  final circlePaint = Paint()
    ..color = Colors.blue.withValues(alpha: 0.2)
    ..style = PaintingStyle.stroke
    ..strokeWidth = 1;
  canvas.drawCircle(center, radius, circlePaint);

  // ---- Evenly spaced points on circle ----
  // Angle for each point: 360¬∞ / n ‚Üí in radians: 2*pi / n
  // Point k is at angle: k * (2*pi / n)
  // x = centerX + radius * cos(angle), y = centerY + radius * sin(angle)
  final pointPositions = <Offset>[];
  for (var k = 0; k < numPoints; k++) {
    final angle = k * (2 * pi / numPoints);
    final x = centerX + radius * cos(angle);
    final y = centerY + radius * sin(angle);
    pointPositions.add(Offset(x, y));
  }

  final pointPaint = Paint()
    ..color = Colors.orange
    ..style = PaintingStyle.fill;
  for (final p in pointPositions) {
    canvas.drawCircle(p, 8, pointPaint);
  }

  // ---- Regular polygon (connect consecutive points) ----
  final polygonPath = Path()..moveTo(pointPositions[0].dx, pointPositions[0].dy);
  for (var i = 1; i < pointPositions.length; i++) {
    polygonPath.lineTo(pointPositions[i].dx, pointPositions[i].dy);
  }
  polygonPath.close();
  final polygonPaint = Paint()
    ..color = Colors.green.withValues(alpha: 0.3)
    ..style = PaintingStyle.fill;
  canvas.drawPath(polygonPath, polygonPaint);
  final polygonStroke = Paint()
    ..color = Colors.green
    ..style = PaintingStyle.stroke
    ..strokeWidth = 2;
  canvas.drawPath(polygonPath, polygonStroke);
}

Trigonometry demo: center point (blue), radius line (grey), circle outline (faint blue), evenly spaced points (orange) using cos/sin, and regular octagon (green fill and stroke).
