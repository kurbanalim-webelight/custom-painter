Sine and cosine curves

A sine wave is the classic smooth wave: it goes up and down in a regular pattern. Mathematically, for a point at horizontal position x, the vertical position is y = centerY + amplitude * sin(angle), where angle increases with x so that one full wave (360¬∞) fits in a chosen width (the wavelength).

In code we usually do something like: loop over x from left to right in small steps; for each x, compute angle = 2 * pi * x / wavelength (so one full wave fits in wavelength pixels), then y = centerY + amplitude * sin(angle). Add each (x, y) to a Path with lineTo (and moveTo for the first point). When you draw that path, you get a smooth sine wave.

  • Amplitude ‚Äî how far the wave goes up and down from the centre line (half the total height of the wave).
  • Wavelength ‚Äî the horizontal distance for one complete cycle (one ‚Äúhump‚Äù and one ‚Äúdip‚Äù).
  • Cosine ‚Äî same idea as sine but shifted by 90¬∞; cos(angle) instead of sin(angle). Use it when you want the wave to start at a peak or trough instead of at the centre.
