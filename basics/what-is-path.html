üìö What is Path?

Path is a class in Flutter (from dart:ui) that represents a sequence of connected lines and curves that form a shape. It's the foundation for drawing custom, complex shapes that can't be created with simple primitives like circles or rectangles.

Think of Path as:

  • A drawing tool that lets you create any shape by connecting points with lines and curves
  • A vector path similar to SVG paths or pen tools in design software
  • A blueprint for complex shapes that you can fill, stroke, or use for clipping
  • A sequence of drawing commands that define how to move and draw on the canvas
üí° Key Concept: Path is used with canvas.drawPath(path, paint) to render custom shapes. You build the path step by step using methods like moveTo(), lineTo(), quadraticBezierTo(), etc., then draw it once.

Common Use Cases

  • Custom shapes: Stars, polygons, arrows, badges
  • Curved designs: Waves, hearts, organic shapes
  • Complex graphics: Logos, icons, decorative elements
  • Clipping regions: Mask areas for images or other content
  • Hit testing: Check if a point is inside a custom shape
  • Animations: Morphing between shapes, path animations

üéØ Essential Path Methods

moveTo(double x, double y)

Purpose: Move the "pen" to a specific point without drawing. This sets the starting point for the next drawing operation.

Common uses: Starting a new path segment, jumping to a different location before drawing.

Think of it as:

  • A painter lifting off his hand from the paper and placing it at a new position
  • A pencil lifting off the paper and moving to a new position
@override
void paint(Canvas canvas, Size size) {
  final path = Path();

  // Move to starting point (no line drawn)
  path.moveTo(50, 50);
}

lineTo(double x, double y)

Purpose: Draw a straight line from the current point to the specified point.

Common uses: Creating polygons, connecting points, drawing straight edges.

@override
void paint(Canvas canvas, Size size) {
  final path = Path();
  final paint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.stroke
    ..strokeWidth = 2;

  final startPoint = Offset(100, 50);

  // Draw a triangle
  path.moveTo(startPoint.dx, startPoint.dy); // Top point
  canvas.drawCircle(
    Offset(100, 50),
    5,
    Paint()..color = Colors.red.withAlpha(180),
  );

  path.lineTo(50, 150); // Bottom left
  canvas.drawCircle(
    Offset(50, 150),
    10,
    Paint()..color = Colors.red.withAlpha(180),
  );

  path.lineTo(150, 150); // Bottom right
  canvas.drawCircle(
    Offset(150, 150),
    15,
    Paint()..color = Colors.red.withAlpha(180),
  );

  path.lineTo(startPoint.dx, startPoint.dy); // Close back to start

  canvas.drawPath(path, paint);
}

quadraticBezierTo(double x1, double y1, double x2, double y2)

Purpose: Draw a quadratic Bezier curve using one control point.

Parameters:

  • x1, y1 - Control point (influences the curve shape)
  • x2, y2 - End point

Common uses: Smooth curves, rounded corners, organic shapes.

@override
void paint(Canvas canvas, Size size) {
  final path = Path();
  final paint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.stroke
    ..strokeWidth = 2;

  // Start point (Green circle)
  path.moveTo(50, 100);
  canvas.drawCircle(Offset(50, 100), 8, Paint()
    ..color = Colors.green
    ..style = PaintingStyle.fill);
  canvas.drawCircle(Offset(50, 100), 8, Paint()
    ..color = Colors.white
    ..style = PaintingStyle.stroke
    ..strokeWidth = 2);

  // Control point (Red circle)
  final controlPoint = Offset(100, 20);
  canvas.drawCircle(controlPoint, 8, Paint()
    ..color = Colors.red
    ..style = PaintingStyle.fill);
  canvas.drawCircle(controlPoint, 8, Paint()
    ..color = Colors.white
    ..style = PaintingStyle.stroke
    ..strokeWidth = 2);

  // End point (Orange circle)
  final endPoint = Offset(150, 100);
  canvas.drawCircle(endPoint, 8, Paint()
    ..color = Colors.orange
    ..style = PaintingStyle.fill);
  canvas.drawCircle(endPoint, 8, Paint()
    ..color = Colors.white
    ..style = PaintingStyle.stroke
    ..strokeWidth = 2);

  // Helper lines (grey dashed) - from start to control, control to end
  canvas.drawLine(Offset(50, 100), controlPoint, Paint()
    ..color = Colors.grey.withAlpha(128)
    ..strokeWidth = 1);
  canvas.drawLine(controlPoint, endPoint, Paint()
    ..color = Colors.grey.withAlpha(128)
    ..strokeWidth = 1);

  // Quadratic Bezier curve
  path.quadraticBezierTo(controlPoint.dx, controlPoint.dy, endPoint.dx, endPoint.dy);

  canvas.drawPath(path, paint);
}

cubicTo(double x1, double y1, double x2, double y2, double x3, double y3)

Purpose: Draw a cubic Bezier curve using two control points for more complex curves.

Parameters:

  • x1, y1 - First control point
  • x2, y2 - Second control point
  • x3, y3 - End point

Common uses: Complex curves, S-shapes, wave patterns, smooth transitions.

@override
void paint(Canvas canvas, Size size) {
  // Define points for cubic Bezier curve
  const startPoint = Offset(50, 150);
  const controlPoint1 = Offset(100, 20); // First control point
  const controlPoint2 = Offset(200, 300); // Second control point
  const endPoint = Offset(250, 150);

  // Draw the cubic Bezier curve
  final path = Path();
  final paint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.stroke
    ..strokeWidth = 3;

  path.moveTo(startPoint.dx, startPoint.dy);
  // cubicTo takes: control1.x, control1.y, control2.x, control2.y, end.x, end.y
  path.cubicTo(controlPoint1.dx, controlPoint1.dy, controlPoint2.dx, controlPoint2.dy, endPoint.dx, endPoint.dy);
  canvas.drawPath(path, paint);

  // Draw reference lines (dashed) to show control points' influence
  final dashedLinePaint = Paint()
    ..color = Colors.grey.withValues(alpha: 0.5)
    ..style = PaintingStyle.stroke
    ..strokeWidth = 1;

  // Line from start to first control point
  final helperPath1 = Path()
    ..moveTo(startPoint.dx, startPoint.dy)
    ..lineTo(controlPoint1.dx, controlPoint1.dy);
  canvas.drawPath(helperPath1, dashedLinePaint);

  // Line from first control point to second control point
  final helperPath2 = Path()
    ..moveTo(controlPoint1.dx, controlPoint1.dy)
    ..lineTo(controlPoint2.dx, controlPoint2.dy);
  canvas.drawPath(helperPath2, dashedLinePaint);

  // Line from second control point to end point
  final helperPath3 = Path()
    ..moveTo(controlPoint2.dx, controlPoint2.dy)
    ..lineTo(endPoint.dx, endPoint.dy);
  canvas.drawPath(helperPath3, dashedLinePaint);

  // Draw reference points with different colors
  // Start point (green)
  final startPointPaint = Paint()
    ..color = Colors.green
    ..style = PaintingStyle.fill;
  canvas.drawCircle(startPoint, 6, startPointPaint);
  canvas.drawCircle(
    startPoint,
    6,
    Paint()
      ..color = Colors.white
      ..style = PaintingStyle.stroke
      ..strokeWidth = 2,
  );

  // First control point (red)
  final controlPoint1Paint = Paint()
    ..color = Colors.red
    ..style = PaintingStyle.fill;
  canvas.drawCircle(controlPoint1, 6, controlPoint1Paint);
  canvas.drawCircle(
    controlPoint1,
    6,

      ..color = Colors.white
      ..style = PaintingStyle.stroke
      ..strokeWidth = 2,

  // Second control point (purple)
  final controlPoint2Paint = Paint()
    ..color = Colors.purple

  canvas.drawCircle(
    controlPoint2,

      ..color = Colors.white

      ..color = Colors.white
      ..style = PaintingStyle.stroke

arcTo(Rect rect, double startAngle, double sweepAngle, bool forceMoveTo)

Purpose: Add an arc (portion of a circle or ellipse) to the path. The arc is part of an oval defined by the bounding rectangle.

Parameters:

  • Rect rect - The bounding rectangle that defines the oval. The arc will be part of this oval. If width == height, it's a circle; otherwise, it's an ellipse.
  • double startAngle - The starting angle in radians where the arc begins. 0 radians = 3 o'clock (right), œÄ/2 = 6 o'clock (down), œÄ = 9 o'clock (left), 3œÄ/2 = 12 o'clock (up).
  • double sweepAngle - How far to sweep the arc in radians. Positive values sweep clockwise, negative values sweep counter-clockwise. 2œÄ = full circle, œÄ = half circle.
  • bool forceMoveTo - If true, starts a new contour (like calling moveTo first). If false, connects from the current point to the arc start with a line.

What happens:

  • The arc is drawn as part of an oval (circle or ellipse) defined by the bounding rectangle
  • If forceMoveTo is false and there's a current point, a line is drawn from the current point to the start of the arc
  • The arc sweeps from startAngle for sweepAngle radians
  • The arc becomes part of the current path contour

Angle reference:

  • 0 radians = 3 o'clock (pointing right ‚Üí)
  • œÄ/2 (1.5708) = 6 o'clock (pointing down ‚Üì)
  • œÄ (3.14159) = 9 o'clock (pointing left ‚Üê)
  • 3œÄ/2 (4.7124) = 12 o'clock (pointing up ‚Üë)
  • 2œÄ (6.28318) = full circle (back to 3 o'clock)

Common use cases:

  • Drawing pie charts and circular progress indicators
  • Creating curved shapes and decorative arcs
  • Drawing partial circles (semicircles, quarter circles)
  • Creating rounded corners and curved borders
  • Gauge dials and circular meters
üí° Key Points:
  • Angles are in radians, not degrees. Use degrees * œÄ / 180 to convert
  • Positive sweepAngle = clockwise, negative = counter-clockwise
  • If forceMoveTo = false, a line connects from current point to arc start
  • If forceMoveTo = true, the arc starts a new contour
  • The rectangle defines the oval's bounds - use square rects for circles
  • For a full circle, use sweepAngle = 2 * œÄ (approximately 6.28318)
@override
void paint(Canvas canvas, Size size) {
  final path = Path();
  final paint = Paint()
    ..color = Colors.blue
    ..style = PaintingStyle.stroke
    ..strokeWidth = 3;

  // Define the bounding rectangle for the oval
  final rect = Rect.fromLTWH(50, 50, 200, 200);

  // Start from a point outside the arc
  path.moveTo(50, 150);

  // Add an arc: start at 0 radians (3 o'clock), sweep œÄ radians (half circle)
  // forceMoveTo = false means connect from current point to arc start
  path.arcTo(
    rect,
    0, // startAngle: 0 radians = 3 o'clock (right)
    3.14159, // sweepAngle: œÄ radians = 180 degrees (half circle)
    false, // forceMoveTo: connect from current point
  );

  // Draw the path
  canvas.drawPath(path, paint);

  // Draw the bounding rectangle (for reference)
  canvas.drawRect(
    rect,
    Paint()
      ..color = Colors.grey.withValues(alpha: 0.3)
      ..style = PaintingStyle.stroke
      ..strokeWidth = 1,
  );

  // Draw center point
  final center = rect.center;

  // Draw start angle indicator (3 o'clock)
  final startPoint = Offset(center.dx + rect.width / 2, center.dy);
  canvas.drawCircle(startPoint, 8, Paint()..color = Colors.green);

  // Draw end angle indicator (9 o'clock, after œÄ sweep)
  final endPoint = Offset(center.dx - rect.width / 2, center.dy);
  canvas.drawCircle(endPoint, 8, Paint()..color = Colors.orange);
}
