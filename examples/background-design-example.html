üìö Overview

This example demonstrates how to create a beautiful background design with elastic waves, gradient circles, and rotated triangles using CustomPainter and various Path methods. The example showcases the use of relativeCubicTo to create smooth elastic waves, RadialGradient and LinearGradient for colorful gradients, and canvas transformations for rotating shapes.

Key Concepts Demonstrated:

  • Creating custom background painters with CustomPainter
  • Using relativeCubicTo to create elastic wave patterns
  • Applying RadialGradient and LinearGradient for colorful effects
  • Using canvas transformations (save, translate, rotate, restore)
  • Building complex paths with relativeLineTo and close
  • Creating decorative shapes (circles, triangles) with gradients

üåä Step 4: Drawing the Top Pink Wave

Create the first elastic wave at the top of the screen using relativeCubicTo.

// ! Top wave
final pinkWavePaint = Paint()
  ..color = Colors.pink.shade600
  ..style = PaintingStyle.fill;

final path = Path();

// Start from top-left
path.moveTo(0, 0);

// Go down a bit
path.lineTo(0, size.height * 0.3);

// Elastic wave using relativeCubicTo
path.relativeCubicTo(
  size.width * 0.25,
  -size.height * 0.15, // control point 1 (relative)
  size.width * 0.75,
  size.height * -0.093, // control point 2 (relative)
  size.width,
  size.width * -0.35, // end point (relative)
);

// Close shape
path.lineTo(size.width, 0);
path.close();

// Draw the path
canvas.drawPath(path, pinkWavePaint);
üí° Explanation:
  • moveTo(0, 0) - Start at the top-left corner
  • lineTo(0, size.height * 0.3) - Draw a vertical line down to 30% of screen height
  • relativeCubicTo - Creates an elastic S-curve using relative coordinates. The two control points create the wave effect
  • lineTo(size.width, 0) - Draw a horizontal line back to the top-right
  • close() - Closes the path to create a filled shape

final circlePaint = Paint()

  Offset(size.width - size.width, size.height),

);
üí° Explanation:
  • RadialGradient - Creates a circular gradient that radiates from the center
  • center: Alignment.bottomLeft - Positions the gradient center at bottom-left
  • radius: 0.8 - Controls how far the gradient extends
  • createShader - Converts the gradient into a shader that can be used with Paint
  • The circle is positioned at the bottom-left corner with a radius of 25% of screen width

üî∫ Step 8: Drawing the Rotated Triangle with Linear Gradient

Create a triangle using relativeLineTo and rotate it using canvas transformations.

//! Bottom center Triangle
final trianglePaint = Paint()
  ..shader = LinearGradient(
    colors: [Colors.purple, Colors.pink],
    begin: Alignment.topLeft,
    end: Alignment.bottomRight,
  ).createShader(Rect.fromCircle(
    center: Offset(size.width * 0.2, size.height * 0.95),
    radius: size.width * 0.18,
  ))
  ..style = PaintingStyle.fill;

final trianglePath = Path();

// Start at center, positioned a bit upwards from the bottom
final startX = size.width * 0.75;
final startY = size.height * 0.812; // A bit upwards from bottom
trianglePath.moveTo(startX, startY);

// Use relativeLineTo to create triangle
// Move to bottom-right
trianglePath.relativeLineTo(size.width * 0.12, size.height * 0.06);
// Move to bottom-left
trianglePath.relativeLineTo(-size.width * 0.24, 0);
// Close the path (back to start)
trianglePath.close();

// Calculate triangle center for rotation
final triangleCenterX = startX;
final triangleCenterY = startY + (size.height * 0.06) / 2;

// Rotate triangle 90 degrees (œÄ/2 radians)
canvas.save();
canvas.translate(triangleCenterX, triangleCenterY);
canvas.rotate(-3.14159 / 2); // 90 degrees in radians
canvas.translate(-triangleCenterX, -triangleCenterY);
canvas.drawPath(trianglePath, trianglePaint);
canvas.restore();
üí° Explanation:
  • LinearGradient - Creates a linear gradient from top-left to bottom-right
  • relativeLineTo - Draws lines relative to the current point, making it easy to build shapes
  • close() - Closes the triangle path
  • canvas.save() - Saves the current canvas state
  • canvas.translate() - Moves the origin to the triangle center
  • canvas.rotate() - Rotates the canvas by -90 degrees (œÄ/2 radians)
  • canvas.restore() - Restores the canvas state, undoing the transformations

class BgDesign extends StatelessWidget {
  const BgDesign({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: CustomPaint(painter: BgDesignPainter(), size: Size.infinite),
    );
  }
}

class BgDesignPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // ! Top wave
    final pinkWavePaint = Paint()
      ..color = Colors.pink.shade600
      ..style = PaintingStyle.fill;

    final path = Path();

    // Start from top-left
    path.moveTo(0, 0);

    path.relativeCubicTo(

      size.height * -0.093, // control point 2 (relative)
      size.width,
      size.width * -0.35, // end point (relative)
    );

    // Close shape
    path.lineTo(size.width, 0);
    path.close();

    // Draw the path first
    canvas.drawPath(path, pinkWavePaint);

    // ! Bottom wave
    final blueWavePaint = Paint()
      ..color = Colors.blue
      ..style = PaintingStyle.fill;

    final path2 = Path();

    // Start from top-left
    path2.moveTo(0, size.height);

    // Go down a bit
    path2.lineTo(0, size.height * 0.95);

    path2.relativeCubicTo(

      size.width,
      size.width * -0.35, // end point (relative)
    );

    // Close shape
    path2.lineTo(size.width, size.height);

        colors: [Colors.purple, Colors.pink],
