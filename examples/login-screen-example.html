üìö Overview

This example demonstrates how to create a beautiful login screen with custom gradient donut shapes in the background using CustomPainter. The example showcases the use of drawDRRect to create donut shapes, SweepGradient for colorful gradients, and MaskFilter.blur for a glowing effect.

Key Concepts Demonstrated:

  • Creating custom background painters with CustomPainter
  • Using drawDRRect to create donut/ring shapes
  • Applying SweepGradient for circular gradients
  • Using MaskFilter.blur for glow effects
  • Layering UI elements with Stack and CustomPaint

üåà Step 5: Creating the SweepGradient

Create a circular gradient that transitions through multiple colors.

// Create gradient
final gradient = SweepGradient(
  center: Alignment.center,
  colors: [
    Colors.blue.shade400,
    Colors.purple.shade400,
    Colors.pink.shade400,
    Colors.orange.shade400,
    Colors.blue.shade400,
  ],
  stops: const [0.0, 0.25, 0.5, 0.75, 1.0],
);
üí° Explanation:
  • SweepGradient creates a circular gradient that sweeps around a center point
  • The gradient transitions: Blue ‚Üí Purple ‚Üí Pink ‚Üí Orange ‚Üí Blue (full circle)
  • stops define where each color appears (0.0 = start, 1.0 = end)
  • Starting and ending with blue creates a seamless loop

üñåÔ∏è Step 6: Setting Up the Paint Object

Configure the Paint object with gradient shader and blur effect.

final paint = Paint()
  ..shader = gradient.createShader(Rect.fromCircle(center: circle1, radius: outerRadius))
  ..style = PaintingStyle.fill
  ..maskFilter = const MaskFilter.blur(BlurStyle.solid, 30);
üí° Explanation:
  • shader - Applies the gradient to the paint (bounded by the outer circle)
  • style - PaintingStyle.fill fills the shape
  • maskFilter - Creates a blur effect with 30-pixel radius for a glowing appearance

üç© Step 7: Creating RRect Shapes for Donuts

Create rounded rectangles (RRect) for both outer and inner circles of each donut.

// Draw donut using drawDRRect
// Create outer RRect (circular by using corner radius = half of size)
final outerRect = Rect.fromCircle(center: circle1, radius: outerRadius);
final outerRRect = RRect.fromRectAndRadius(outerRect, Radius.circular(outerRadius));

// Create inner RRect (circular by using corner radius = half of size)
final innerRect = Rect.fromCircle(center: circle1, radius: innerRadius);
final innerRRect = RRect.fromRectAndRadius(innerRect, Radius.circular(innerRadius));

final outerRect2 = Rect.fromCircle(center: circle2, radius: outerRadius);
final outerRRect2 = RRect.fromRectAndRadius(outerRect2, Radius.circular(outerRadius));
final innerRect2 = Rect.fromCircle(center: circle2, radius: innerRadius);
final innerRRect2 = RRect.fromRectAndRadius(innerRect2, Radius.circular(innerRadius));
üí° Explanation:
  • By setting corner radius equal to the radius, we create perfect circles
  • Each donut needs an outer RRect (large circle) and inner RRect (hole)
  • We create two sets: one for circle1 and one for circle2
  • drawDRRect will draw the outer shape and cut out the inner shape

üéØ Step 8: Drawing the Donut Shapes

Use drawDRRect to draw both donut shapes on the canvas.

// Draw donut shape using drawDRRect
canvas.drawDRRect(outerRRect, innerRRect, paint);
canvas.drawDRRect(outerRRect2, innerRRect2, paint);
üí° Explanation:
  • drawDRRect draws a double rounded rectangle (donut shape)
  • It draws the outer shape and cuts out the inner shape
  • Both donuts use the same paint (gradient and blur), creating a consistent glowing effect
  • The first donut appears at top-right, the second at bottom-left

//! Donut Background Painter
/// Draws a gradient donut shape in the background
class DonutBackgroundPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final circle1 = Offset(size.width, size.height * 0.3);
    final circle2 = Offset(size.width - size.width, size.height * 0.95);
    final outerRadius = math.min(size.width, size.height) * 0.45;
    final innerRadius = outerRadius * 0.4;

    // Create gradient
    final gradient = SweepGradient(
      center: Alignment.center,
      colors: [
        Colors.blue.shade400,
        Colors.purple.shade400,
        Colors.pink.shade400,
        Colors.orange.shade400,
        Colors.blue.shade400,
    ],
      stops: const [0.0, 0.25, 0.5, 0.75, 1.0],
  );

    final paint = Paint()
      ..shader = gradient.createShader(Rect.fromCircle(center: circle1, radius: outerRadius))
      ..style = PaintingStyle.fill
      ..maskFilter = const MaskFilter.blur(BlurStyle.solid, 30);

    // Draw donut using drawDRRect
    // Create outer RRect (circular by using corner radius = half of size)
    final outerRect = Rect.fromCircle(center: circle1, radius: outerRadius);
    final outerRRect = RRect.fromRectAndRadius(outerRect, Radius.circular(outerRadius));

    // Create inner RRect (circular by using corner radius = half of size)
    final innerRect = Rect.fromCircle(center: circle1, radius: innerRadius);
    final innerRRect = RRect.fromRectAndRadius(innerRect, Radius.circular(innerRadius));

    final outerRect2 = Rect.fromCircle(center: circle2, radius: outerRadius);
    final outerRRect2 = RRect.fromRectAndRadius(outerRect2, Radius.circular(outerRadius));
    final innerRect2 = Rect.fromCircle(center: circle2, radius: innerRadius);
    final innerRRect2 = RRect.fromRectAndRadius(innerRect2, Radius.circular(innerRadius));

    // Draw donut shape using drawDRRect
    canvas.drawDRRect(outerRRect, innerRRect, paint);
    canvas.drawDRRect(outerRRect2, innerRRect2, paint);
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}
